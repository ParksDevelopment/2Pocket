## Introduciton

The 2Pocket is a pocket keyboard originally created for the [Pocket Keyboard Design Contest](https://chrischrislolo.github.io/orthoLabLogs/pocket-keyboard-design-contest-00.html). As such it was designed to meet the criteria of the contest as closely as possible while still being a practical keyboard for daily use.
